Java Concurrent Animated  
 Visualizing the Java Concurrent API  
 
  • Ctrl-R resets the current animation.
 
 
  • Ctrl-S pauses/resumes the current animation.
 
 
  • Ctrl-H toggles the menu bar on the left.
 
 
  • Drag inside snippet panel to resize it
 
 
  • Use Up and Down arrows to page
    between the slides.
